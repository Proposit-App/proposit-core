# 0.5.10

## Bug Fixes

- **`normalizeChecksumConfig` handles native JSON round-trips:** `JSON.stringify(new Set(...))` produces `{}` (an empty plain object), not an array. The normalizer from 0.5.9 only handled arrays, causing `new Set({})` to throw. Now handles three cases: Set pass-through, array-to-Set conversion (custom serializers), and plain-object fallback to empty Set (native `JSON.stringify`).
